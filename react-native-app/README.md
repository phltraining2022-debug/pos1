# Kara POS Mobile

Expo-based React Native starter that mirrors the web dashboard layout in `react-app`.

## Run

```bash
npm install
npm run start
```

## Notes

- Built mobile-first for staff tablets and phones.
- Shares the same operational concepts as the web UI: rooms, orders, quick actions, and priorities.
- The file is intentionally presentational so business logic can be split into shared data modules later.
